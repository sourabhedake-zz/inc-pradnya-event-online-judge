TestCase Constraints
1 <= t <= 10
1 <= n,m <= 100

Alternating

Problem Statement
You are given a grid of n*m blocks, i.e. n rows and m columns. (Rows and columns are 1-indexed).
Each block of the grid contains either of the 2 symbols, '(' or ')'
A position (i,j) denotes the ith row and the jth column.
Initially you are at (1,1) and you need to reach (n,m).
On any block (i,j), you can have either if the 2 moves:
1. (i+1,j), if i < n
2. (i, j+1), if j < m
You need to determine whether it is possible to traverse to get an alternating balanced sequence.
Note: An Alternating balanced sequence is of the pattern ()()()....()



Input and Output Format
Input Format:
The first line contains t, denoting the number of test cases.
Each test case consists of 2 space separated integers n and m.
The next n lines of each test case contains m space separated symbols, either of '(' or ')',
denoting the blocks of the grid.

Output Format:
For each test case, output on each line "YES", if it is possible to get alternating balanced sequence, else output "NO"

Sample Input & Output with Explanation
Sample Input:
2
1 1
(
1 2
( )

Sample Output:
NO
YES
